# Migração Completa dos Botões, Links e Conteúdo Remanescente

Versão: 1.0

## OBJETIVO PRINCIPAL

Finalizar a migração de todos os botões, links, páginas, documentos e conteúdos existentes no portal antigo para o novo portal.

**IMPORTANTE:** O novo portal encontra-se em produção e completamente funcional.

Não é permitido alterar:
- Layout
- Estrutura visual
- Organização da Home
- CSS existente
- Componentes existentes
- Banco de dados atual
- Painéis existentes
- Funcionalidades atuais
- Sistema administrativo já implementado

Todo o trabalho consiste exclusivamente em complementar o conteúdo que ainda não foi migrado.

## SITE DE ORIGEM

https://curriculo.sedu.es.gov.br/curriculo/

## DIRETRIZES

- Comparar automaticamente portal antigo × portal novo.
- Importar apenas itens inexistentes.
- Nunca duplicar registros.
- Nunca sobrescrever conteúdo existente.
- Nunca remover conteúdo existente.
- Criar automaticamente categorias novas quando necessário para itens sem categoria correspondente.
- Todo conteúdo importado deverá ficar disponível no painel administrativo já existente.
- Links que ainda apontam para o portal antigo devem permanecer inalterados, pois o portal antigo continuará ativo temporariamente como subdomínio.
- Gerar log completo da importação (categorias, botões, links, páginas, documentos, duplicidades e erros).

## REGRAS DE SEGURANÇA

Nunca:
- alterar layout
- alterar CSS
- alterar JavaScript
- alterar componentes
- alterar menus
- alterar páginas existentes
- alterar URLs existentes
- alterar banco de dados existente de forma destrutiva
- excluir conteúdo
- sobrescrever registros existentes

## RESULTADO ESPERADO

- Todo o conteúdo remanescente do portal antigo disponível no novo portal.
- Estrutura visual e funcional preservada integralmente.
- Painel administrativo existente gerenciando também os novos conteúdos importados.
- Migração transparente e segura.
